Experience

ML & Web Development

During my tenure, as a project intern in the Software Department at ISRO-RRSC, Bangalore, My team members and I worked on an ongoing concern of RRSC for the past 6 months, under the assistance of scientists, and successfully developed a prototype for detecting a lake water quality using ML & Image processing and a web site to host our data.

Product Development

The design and development of the prototype BUOY (a device capable of floating and moving in the desired direction on water)handled by the Hardware Team. Comprised of 4 major parameters pH, Turbidity, TDS, Temperature sensors, a ESP32 CAM to capture images for Image Processing, Wifi module to to transfer the collected data, etc.

Analytical Phase

During the analytical phase of our project, we employed machine learning (ML) and image processing techniques. In ML, we trained and evaluated 10 regression and classification models to predict the water quality index (WQI). The model with the highest accuracy was then used to predict the actual WQI from the data provided by the hardware team.

Web Development

In web development, We have used Python Flask framework as the backend, the phpMyAdmin database management tool via the XAMPP local web server, and custom CSS, HTML, and the Bootstrap framework for the frontend.

Integration Phase

In the integration phase, we used a WiFi module to transfer the collected data to the cloud. We also used Pickle and Streamlit to integrate the analytical phase models to a web application. Additionally, we developed a navigation system for the buoy using JavaScript and the Leaflet library.
